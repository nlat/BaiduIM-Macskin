# 百度 Mac 输入法皮肤

目前只完成了单行模式，还没有双行模式，没有状态栏。

因为自己用不着，后期也无开发计划。

如君需要，下载 bdskk 文件，即可。

如需修改，下载 源文件，修改打包为 bdskk 格式即可。

只适用于 Mac 版百度输入法，其他系统没有测试。
